# GestureSense AI

### Real-Time Motion Recognition Using Smartphone Sensors

GestureSense AI is a web-based gesture recognition platform that uses smartphone motion sensors and Edge AI to classify gestures in real time.

## Features

- Real-time accelerometer data acquisition
- Circle gesture recognition
- Wave gesture recognition
- Idle state detection
- Edge Impulse WASM inference engine
- Firebase Hosting deployment
- Mobile-friendly responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Edge Impulse
- Firebase Hosting
- Smartphone Motion Sensors
